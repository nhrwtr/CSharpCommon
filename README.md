## Description

C#であったら便利そうな基本的な機能をまとめた汎用ライブラリです。
独自性もなく誰でも思いつく処理だと思うので、パブリックドメインとします。
問題がある場合は、issueか下記ライセンスに記載のメールアドレスから連絡ください。

It is a generic library of basic functions useful in C#.
